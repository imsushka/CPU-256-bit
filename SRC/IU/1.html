Я подумал подумал и решил, будет у меня мало команд. Типа RISC процессор.

Выделил под команды 6 бит, то есть  всего 64 команды.

  1. Load - загрузить регистр/память из регистра/памяти/команды
  2. Swap - обменять 2 регистра/памяти
  3. Comp - сравнить
  4. And / Or / Xor - логические команды
  5. Slr / Sll / Sar - сдвиги вправо/влево логические и арифметические
  6. Ror / Rol / Rorc / Rolc - сдвиг с переносом
  7. Add / Addc / Adds / Sub / Subc / Subs - сложение/вычитание простое/с переносом/со знаком
  8. Mul / Muls / Div / Divs / Mod - умножение/деление/остаток
  9. Jmp / Call / Int / Ret - команды переходов
  10. Push / Pop - стек
  11. BitSet / BitReset / BitTest - работа с битами

И хоть эти команды заняли 32 позиции, но не получилось у меня RISC.
